Aula: Desenvolvimento Web com Javascript

O JavaScript é uma linguagem interpretada de tipagem dinâmica, ou seja, é possível mudar o tipo de uma variável em tempo de execução. Permite uma melhor interatividade de um site, trabalhando junto com HTML e CSS. O código JavaScript é integrado ao HTML5, para isso, utiliza-se a tag <script>.
O JavaScript permite a criação de jogos em 2D e 3D com ligação com banco de dados

Arquitetura Cliente-Servidor

JavaScript roda do lado do cliente, ou seja, nos navegadores de internet.

Funções

As funções são formas de encapsular funcionalidades que se deseja utilizar.
A função guarda um determinado procedimento para que, sempre que você precisar dele novamente, possa chamá-lo apenas pelo nome da função. Isso evita redundância no código e diminui o seu tamanho. EX: Alert();

  • Escopo Global: A variável é declarada pra poder ser usada em todo o código.
  • Escopo de Função: A variável é declarada dentro de uma função.

Variável de bloco: São criadas em blocos de instruções como IF, FOR, While, etc.

DOM

Document Object Model ou DOM é usado para acessar objetos na página HTML. Ele possibilita realizar alterações na estrutura de um documento, resolvendo parcela significativa de problemas do lado do cliente. Diminui o trafego de dados entre o cliente e o servidor pois você tem a possibilidade de modificar apenas um campo da página se desejar sem necessitar fazer novas requisições ao servidor.
Ele permite que o usuário faça customizações na página, como alteração de layouts. Para isso usamos alguns métodos como:
  • GetElementById(): retorna elemento que estiver com Id informado.
  • GetElementByClass(): Retornar um HTML Collection de todos os elementos que contiverem o nome da classe informada.
  • GetElementByTagName(): Retorna os elementos contendo a tag name informada

Além dos métodos, podemos usar Eventos, que aumentam as possibilidades de interatividade. Alguns exemplos:
  • onclick: usuário clica em um elemento
  • onchange: mudança em elemento HTML
  • onload: página é carregada
