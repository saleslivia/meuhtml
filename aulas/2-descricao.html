HTML, CSS e JavaScript


De maneira simples, podemos difinir HTML, CSS e JavaScript da seguinte maneira:

  • O HTML, HyperText Markup Language, é a estrutura da página. Como se fosse um corpo (esqueleto).
  • O CSS, Cascading Style Sheets, é o que dá estilo aos códigos, permitindo criar a formatação desejada e gerar atratividade . Como se fosse a roupa que a pessoa utiliza.
  • O Javascript é uma liguagem de programação, é ele que dá movimento e ação, criando a interação da página. Como se fosse o espírito do corpo.

Estrutura HTML:

            <!DOCTYPE html>
            <html lang="pt-br">
            <head>
            <meta charset="UTF-8">
                <meta http-equiv="X-UA-Compatible" content="IE=edge">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">
                <title>Document</title>
            </head>
            >body>
        
            </body>
            </html>
No exemplo acima:
  • <!DOCTYPE html>: Indicar que o documento está em HTML 5.
  • <html lang="pt-br">: indicar que o conteúdo do site será em Português do Brasil.
  • <head>: É a área de configuração.
  • <meta charset="UTF-8">: Significa que o site é conpatível com acentuação.
  • <meta http-equiv="X-UA-Compatible" content="IE=edge">: É exclusiva para IE, ela pode configurar a página para ser renderizada como em outra versão do Internet Explorer.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Significa que o site vai ocupar 100% de tela e ser visualizado com 100%, sem zoom
  • <meta http-equiv="refresh" content="30">: Atualiza a página a cada 30 segundos
  • <meta name="robots" content="noindex,nofollow">:Essas metatags controlam o comportamento de rastreamento e indexação dos mecanismos de pesquisa.
  • <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">: Aqui você coloca o Fivicon
  • <title>Document</title>: Título da página no navegador
  • <body>: É onde fica o conteúdo do site.
  • Temos outras diversas Metas TAG's usadas para inserir fivicons, arquivos de fonte, de estilo etc

Estrutura em CSS

            .codigo{
                background-color: #fff;
                color: #000;
                padding-top: 5px;
                width: 700px;
            }
No exemplo acima:
  • .codigo é um seletor.
  • background-color: #fff;, é a declaração.
  • background-color: é a propriedade e
  • #fff, é o valor (neste caso em hexadecimal).
