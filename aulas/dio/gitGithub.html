- Media Queries

As media queries são regras do CSS que nos permitem aplicar determinadas formatações apenas em condições específicas (aplicação condicional de estilos), como o comportamento de uma página, dependendo do dispositivo/midia (media types) que a estiver exibindo. Junto com as Media Type, formam a base de todo layout responsivo.
