Condições em JavaScript


Condição é a estrutura de condição a ser verificada para decidir se as linhas do bloco de código serão executadas ou não. Caso a condição seja falsa, o bloco de código 1 não deve ser executado, mas o bloco de código 2 sim.

  • Condição Simples: IF 
                var vel=50
            console.log('a velocidade do seu carro é '+ vel +'Km/h')
            if(vel>=60){
                console.log('Você utrapassou o limite de velocidade e será multado')
            }
            console.log('Use sempre sinto de segurança!')
  • Condição Composta: IF e ELSE 
                var pais='França'
            console.log('Vivendo em '+pais)
            if (pais!='Brasil'){
                console.log('Você é estrangeiro!')
            }else{
                console.log('Você é brasileiro!')
                }
